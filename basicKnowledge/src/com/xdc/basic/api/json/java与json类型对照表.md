# java与json类型对照表  

| JSON Type            | Java Type                                         |  
| -------------------- | ------------------------------------------------- |  
| object               | LinkedHashMap\<String,Object\>                    |  
| array                | ArrayList\<Object\>                               |  
| string               | String                                            |  
| number (no fraction) | Integer, Long or BigInteger (smallest applicable) |  
| number (fraction)    | Double (configurable to use BigDecimal)           |  
| boolean              | Boolean                                           |  
| null                 | null                                              |
